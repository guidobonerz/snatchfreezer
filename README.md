# SnatchFreezer (Beta)
SnatchFreezer is a project for Highspeed photography

Runs on Windows, Mac and Linux

This project also contains all required software and PCB layouts including:
- JavaFX - Controller Frontend
- Arduino Sketch
- Eagle PCB
